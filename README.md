# rustqlite

![Rust](https://img.shields.io/badge/rust-stable-000000?logo=rust&logoColor=white)
![License: MIT](https://img.shields.io/badge/license-MIT-blue.svg)
![Status](https://img.shields.io/badge/status-educational-orange)

An educational, minimal Rust re-implementation of a small subset of SQLite.  
This project follows the “Build Your Own SQLite From Scratch” tutorial by cstack:  
https://cstack.github.io/db_tutorial

## Goals

- Learn how a simple database works internally (pager, B-Tree, serialization, etc.)

## Non-goals

- Production use
- Full SQLite compatibility
- Stable storage format or API
